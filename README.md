# GrowEasy AI CSV Importer

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Node.js
- Express
- Gemini AI

## Features

- CSV Upload
- Drag & Drop
- CSV Preview
- AI-powered CRM field extraction
- Responsive Dashboard
- Import Statistics
- Search Imported Leads

## Setup

Frontend

npm install
npm run dev

Backend

npm install
npm start
